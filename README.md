# 🚩 Unfriendly Fitness App (Agent Benchmark)

**Unfriendly Fitness App** is a deliberate "Anti-Pattern" reference application. It is built using modern Android architecture (Navigation 3, Compose Adaptive APIs, and Material 3) but purposely ignore edge-to-edge.

## 🎯 Purpose
The goal of this project is to provide a standardized environment to test how effectively AI coding agents can identify and refactor an existing codebase to support **Edge-to-Edge (E2E)** and proper **Window Inset** handling in Android 15+.
